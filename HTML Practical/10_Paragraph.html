HTML elements are the building of HTML pages. With HTML constructs, images and other objects such as interactive forms may be embedded into the rendered page.
HTML provides a means to creat structured documents by dentoing stuctural semantics for text such asheadings, paragraphs, lists, links, quotes and other items.
HTML elements are delineated by tags, written using angle brackets.Tags such as <img/> and <input/> directly introduce content into the page
Other tags such as <p> surrounf and provide information about document text and many include other tags as sub-element.
Browsers do not display the HTML tags but use them to interpret the content of the page.
